Loops In JS

While Loop

In Javascript , a while loop is a control flow statement that allows you to repeatedly execute a block of code as long as a specified condition is true.

Do While Loop

In Javascript, a do while is control flow statement that repeatedly executes a block of code as long as a specified condition evaluates to true.

For Loop

for loop runs through the block of code a predetermined number of times.And it works well when you know exactly how many iteration you need.

Types of for loop:
1.for/in:loops through the properties of an object
2.for/of:loops through the properties of string and arrays
