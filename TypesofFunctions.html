Arrow Function

Arrow function is a consice way of writing the function

An Anonymous function in Javascript is a function without a name.It is often used in situations where a function is needed temporarily,such as passing a function as an agrument to another function

In Javascript,variables can have either global or local scope.
Global scope means the variable is accesible throughout the entire program,
local scope means the variable is only accesible within a spcecific block or function

A closure is the combination of a function bundled together (enclosed) with refrerence to its surronding state(the lexical environment).In other words. a closure gives you access to an outer function's scope from an inner function

In Javascript,a pure function is a function that,given the same input, will always return the same output and has no side effects.(does not modify the state)
