Example of Event bubling,capturing and delegation

Event Bubling:In the event bubbling phase,the event starts from the target element that triggered the event and bubbles up through its ancestor in the DOM hierarchy. Most modern browser follow the event bubbling model by default

This is a paragraph

Capturing:In the event capturing phase,the event starts from the top-level ancestor and trickles down to the target element that triggered the event.You can enable event capturing by setting the third parameter of addEventListener to true

paragraph

Event delegation:Event delegation is a technique in javascript where a single event listener is placed on a common ancestor (usually a parent) of multiple elements,and it leverages event bubbling to handle events for all the child elements
